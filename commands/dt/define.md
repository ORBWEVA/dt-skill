Run Stage 2 (Define) from the Design Thinking skill at `~/.claude/skills/design-thinking/SKILL.md`.

1. Synthesize insights from empathy work (or gather them now if not done yet)
2. Write a "How Might We" statement: HMW [action] for [user] so that [outcome] without [constraint]?
3. Define 3-5 measurable success criteria
4. Scope check — is this too broad, too narrow, or already solved?

Deliver: HMW statement + success criteria + scope assessment.
