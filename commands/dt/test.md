description: Create test plans, feedback frameworks, and success metrics for prototypes
---

Run Stage 5 (Test) from the Design Thinking skill at `~/.claude/skills/design-thinking/SKILL.md`.

1. Define test plan per prototype: hypothesis, method, audience, duration, success/failure thresholds
2. Identify what to observe: task completion, time on task, error rate, emotional response, unprompted feedback
3. Create feedback capture framework: Keep / Fix / Add / Remove
4. Decision framework: clear success → ship; partial → iterate (Stage 3/4); clear failure → pivot (Stage 2); surprise → explore (Stage 1)

Deliver: Test plan + observation checklist + decision framework + recommended next action.
