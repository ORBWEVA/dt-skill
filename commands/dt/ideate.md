Run Stage 3 (Ideate) from the Design Thinking skill at `~/.claude/skills/design-thinking/SKILL.md`.

1. Diverge — generate 8-12 ideas without judgment using constraint-breaking prompts
2. Challenge 3-5 assumptions baked into current thinking
3. Prioritize using Impact/Effort matrix (Quick Wins, Big Bets, Fill-ins, Money Pits)
4. Select 2-3 candidates with rationale and key risk per candidate

Deliver: Idea list + assumption challenges + prioritization matrix + selected candidates.
