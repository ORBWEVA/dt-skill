name: dt:prototype
description: Generate specs, wireframes, or code prototypes at the right fidelity level
---

Run Stage 4 (Prototype) from the Design Thinking skill at `~/.claude/skills/design-thinking/SKILL.md`.

1. Choose fidelity level (low/medium/high) based on what assumption needs validating
2. For UI/UX: describe layout, user flow, key states, content hierarchy
3. For systems/architecture: describe data flow, API contracts, state management, integrations
4. For dashboards: what questions it answers, top 3-5 metrics, available actions, info hierarchy
5. Build constraints: testable in 30 min, reveals key assumption, throwaway-friendly

Deliver: Prototype artifact + what assumption it tests + how to test it.
