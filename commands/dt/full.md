Run the full Design Thinking skill from `~/.claude/skills/design-thinking/SKILL.md`.

Walk through ALL 5 stages sequentially for the current project or a specified feature:

1. **Empathize** — Build empathy maps for each user type, identify pain points
2. **Define** — Synthesize a HMW statement + success criteria
3. **Ideate** — Divergent brainstorm, assumption challenges, impact/effort matrix, select 2-3 candidates
4. **Prototype** — Generate specs/wireframes/schemas for selected candidates
5. **Test** — Create test plans with hypotheses, success/failure criteria, decision framework

If the user specified a feature or area, focus on that. If not, ask what they want to design-think about.

Read the project's master doc and relevant code first to ground the exercise in reality. Reference the full SKILL.md for stage details and output formats.
